# Commands Reference

## /tts join
Join your current voice channel and read messages from that voice channel's chat.
**Usage**: `/tts join`

## /tts leave
Leave the voice channel.
**Usage**: `/tts leave`

## /tts say
Read a message aloud.
**Usage**: `/tts say message:<text>`

## /settings view
View current TTS settings.
**Usage**: `/settings view`

## /settings set
Update a TTS setting.
**Usage**: `/settings set key:<setting> value:<value>`
